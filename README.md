**Aplicación Django para gestionar autos, sucursales y ventas.**

## Descripción
Este proyecto es una aplicacion web desarrollada con Django que permite gestionar informacion de autos, sucursales, ventas
- Registrar y listar **autos**.
- Crear y buscar **sucursales**.

## Funcionalidades
- **Página de inicio** con accesos rápidos.
- **Crear autos** con datos básicos (modelo, marca, año, color, precio).
- **Buscar autos** por nombre de modelo.
- **Crear sucursales** con datos basicos. (ciudad, pais)-
-  **Buscar sucursales** por nombre de ciudad
